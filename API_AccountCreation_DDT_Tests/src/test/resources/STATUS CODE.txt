200
201 - на POST запросы
204 - на PUT, PATCH. DELETE 
302 - redirect, login/logout
400 - посланы не валидные данные 
401 - не авторизован
404 - нет чего-то, например нет клиента, которого ты пытаешься удалить
413 - слишком большой файл (в галерее или при добавлении нового клиента)
415 - неподдерживаемый формат файла (например .exe)
422 - данные валидны, нo обработать их не получилось (например дубликат мейла в регистрации)
500 - общая серверная ошибка
502 - отвалился какой-то сервис (у меня несколько бекендов). Надо сделать retry
503 - я смог определить, что пизда базе. Надо сделать retry - в ответе придет header Retry-After: 1 - через сколько секунд надо сделать retry
504 - бекенд завис, не надо делать retry
